How to set up a card printing work station from first principles

ANDiS URL - https://andis.warwick.ac.uk/cms/index.jsp?app=/config.xml 

Software pack :  https://files.warwick.ac.uk/leightonjoskey/browse/Card+Printing+Pack

1)      Install standard desk side image onto PC's.

SNAPE/STAFF/FINAL/SYSPREP.GHO

2)      Uninstall IE7 to leave IE6 (refer to desk side for how to do this (files included in this software pack (ZIP FILE - "IE7Remover"). Once unzipped go to the folder IE7 and then Spunist and runs spuninst.exe best to copy the files to the D: drive first).

3)      OPTIONAL (if using webcams for photo acquisition) Install Logitech webcam using drivers supplied with webcam. (or from www.logitech.com)

4)      OPTIONAL (if using serial port based card printers) Install the StarTech USB-serial port dongle using drivers supplied with dongle. (or from www.startech.com)

5)      Install DataCard printer  as a local printer

a.       For serial port based printers install files are in this software pack (ZIP FILE - "xp-2000 drivers for Datacard sp55")

b.      For USB based printers use the CD supplied with printer or download latest drivers from http://www.datacard.com/id-system-support-and-drivers (specifically as of Aug 2009: http://www.datacard.com/id-card-printer-support-and-drivers/sp55-card-printer#drivers-and-downloads)

c.       Restart PC

6)      Install gem access active control

a.       Extract ?GEM access control ActiveX if config CLCbay control doesn?t appear.zip? from the included software pack onto local hard drive.

b.      Navigate into folder to ?install? and double click.

c.       Batch file will run and gem access control will be registered.

d.      Leave ?GEM access control ActiveX if config CLCbay control doesn?t appear? folder where it is.

7)      Configure ANDiS card printing;

                                I.            Log into Card Production -> Generate Production Order

                              II.            Choose a card family

                            III.            Choose any card (NOTE: we won't actually print the card so choice is arbitrary)

                            IV.            Click IE6 security bar at top of screen (the bar that queries if you want to download the active x control) and select install active X component (NOTE: OK the ANDiS dialogue box that says GEM access is not available). This will log you out of your ANDiS session.

                              V.            Log into Card Production -> Generate Production Order

                            VI.            Choose a card family

                          VII.            Choose any card (NOTE: we won't actually print the card so choice is arbitrary)

                        VIII.            Click install on the windows dialogue box.

                            IX.            Click OK on the ANDiS dialogue box that says gem access can't be installed.

                              X.            Click on "configure printer"

                            XI.            Physical printer should be set to "Smart Driver"

                          XII.            Printer type should be set to "DataCard"

                        XIII.            Bay names should be "CLCbay" (NOTE: capitalisation is important)

                        XIV.            All 4 boxes under options should be unchecked

                          XV.            Click OK

                        XVI.            Click on "Configure CLCbay"

a.       Serial based printer ? set serial port option and auto detect reader

b.      USB based printer ? set to USB option

8)      OPTIONAL (if using ANDiS photo studio) Configure ANDiS photo studio;

                                I.            Log in to ANDiS and navigate to photo -> photo studio

                              II.            Click on photo capture icon - click IE6 security bar at top of screen (the bar that queries if you want to download the active x control) and select install active X component. This will log you out of your ANDiS session.

                            III.            Log in to ANDiS and navigate to photo -> photo studio

                            IV.            Click on photo capture icon - click IE6 security bar at top of screen (the bar that queries if you want to download the active x control). This will install the active x component (note you may need to OK a windows security dialogue checking the active x component is allowed).

Important Information

 

 ANDiS card printing REQUIRES IE6 and will fail if you use IE7 or IE8

 All other ANDiS functions will work under IE7 or IE8 as well as IE6

 If you are using a serial port based card printer, cancel out of ANDiS and close down IE6 after configuring the CLCbay. The serial port is locked by this process and any attempt to print directly after changing the CLCbay settings from the configure CLCbay dialogue box without restarting IE6 will result in a failed print.

 The other zip files included in the software pack are for atypical installation situations - please refer to the staff and membership application management service team in ITS for further advice BEFORE using these files.

The latest (at time of publication SP55 series 4 driver is in the software pack under the name: DATACARDSP_XP_VISTA_DRIVER_27_08_2009.exe
